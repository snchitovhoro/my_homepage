1. What is the main value of Java Server Pages?

    Java Server Pages facilitates the creation of web pages by mixing of dynamic and static content in a platform independent way. JSPs have access to the entire family of java APIs

  2. How are JSP pages related to servlets?

    Java Server Pages are complementary to servlets, a JSP page is compiled into a servlet

  3. How are JSP pages related to HTML?

    JSP consists HTML markup embeded in JSP tags. When the JSp file is processed on the server, the HTML is rendered as a web page
